## ملخص المشروع

موقع **MyCar - مركز متكامل للعناية بالسيارات** بهوية بصرية نظيفة: خلفية بيضاء، نصوص سوداء/رمادية، ذهبي (#D4AF37) للأيقونات والأسعار، خط Cairo، تصميم Flat. كل قسم رئيسي صفحة مستقلة، متجاوب بالكامل مع شريط تنقل سفلي على الجوال.

## نطاق المرحلة 1 (هذه الرسالة)

سأبني واجهة الموقع كاملة + بنية البيانات السحابية + بيانات أولية. السلة/الدفع/لوحة التحكم الكاملة في رسائل تالية حتى لا نستهلك كريدتس زيادة وتظل كل مرحلة قابلة للمراجعة.

## ما سيتم بناؤه

### 1) الهوية البصرية والتصميم
- خط **Cairo** من Google Fonts، اتجاه RTL على الـ `<html>`.
- نظام ألوان في `src/styles.css` بتوكنز (`--gold`, `--ink`, `--ink-soft`, `--hairline`) بصيغة oklch.
- مكونات بطاقة منتج، زر ذهبي، أيقونة دائرية، badge السلة، شريط علوي متحرك (marquee).

### 2) الصفحات (Routes)
- `/` الرئيسية: top bar متحرك → header لاصق → بنر فيديو (placeholder + صورة مولّدة) → أيقونات أقسام دائرية أفقية → Hot Deals (3 بكجات) → Best Sellers (4 منتجات) → فوتر كامل.
- `/shop` المتجر: شبكة 4 أعمدة (2 على الجوال) + فلتر بالأقسام.
- `/product/$id` تفاصيل المنتج: معرض صور + سعر + وصف + زر "أضف للسلة" (واجهة فقط الآن) + زر واتساب + منتجات مشابهة.
- `/services` فهرس الخدمات الـ11 + صفحة لكل خدمة `/services/$slug` (عنوان، وصف، باقات، معرض، نموذج حجز يرسل لواتساب).
- `/vip` صفحة الفريق المتنقل + نموذج طلب.
- `/offers` البكجات + عداد تنازلي + كوبونات (MYCAR10 / VIP20).
- `/about` و `/contact` (مع موقع حدة + ساعات العمل + الأرقام).
- `/admin` صفحة دخول بكلمة `zain20267731` + هيكل أولي للوحة (الإدارة الكاملة في المرحلة التالية).
- صفحة 404.

### 3) Header / Footer / Bottom Nav
- **Top bar** ذهبي بنص متحرك ترحيبي.
- **Header** أبيض لاصق: لوجو MY CAR + اسم "احمد النود — المدير العام" تحت الشعار، مربع بحث، أيقونات (مفضلة، حساب، سلة بعداد).
- **Footer**: شعار + روابط سريعة + تواصل (777472412 / واتساب 733340222 / حدة - شارع صفر) + ساعات العمل + شعارات المحافظ السبع + سطر "تطوير: صديق الزين 780687704 — جميع الحقوق محفوظة".
- **Bottom Nav** على الجوال فقط: الرئيسية / المتجر / الخدمات / السلة / الحساب.

### 4) Lovable Cloud (الباك إند السحابي)
سأفعّل Cloud وأنشئ هذه الجداول مع RLS:
- `categories` (أقسام المتجر السبعة)
- `service_categories` (أقسام الخدمات الـ11)
- `products` (اسم، سعر، وصف، قسم، صور[]، فيديو، تقييم، is_bestseller)
- `packages` (بكجات العروض)
- `wallets` (المحافظ السبع: اسم، رقم، أيقونة)
- `site_content` (نصوص الشريط العلوي، البنر، about... قابلة للتعديل لاحقاً من اللوحة)
- `orders` (هيكل جاهز للمرحلة التالية)
- bucket `media` عام لرفع صور المنتجات لاحقاً

سأملأ الجداول ببيانات أولية: الأقسام السبعة للمتجر، الـ11 خدمة، المحافظ السبع بأرقامها، البكجات الثلاثة، و8-10 منتجات تجريبية بصور مولّدة.

### 5) صور مولّدة
- صورة بنر هيرو (سيارة فاخرة في كراج عناية، إضاءة دراماتيكية).
- 8-10 صور منتجات على خلفية بيضاء (شامبو، معطر، إكسسوار، إلخ).
- صور خدمات (PPF، نانو سيراميك، تنجيد).

### 6) ملاحظات
- روابط واتساب جاهزة بصيغة `https://wa.me/967733340222?text=...`.
- جميع البيانات (أرقام، عناوين، خدمات، بكجات، كوبونات، محافظ) كما وردت في الطلب بدقة.
- لوحة التحكم محمية بكلمة المرور المطلوبة، تُحفظ الجلسة في `sessionStorage`.

## ما **لن** يدخل في هذه الرسالة (المرحلة 2 لاحقاً)
- صفحة السلة الكاملة + checkout + اختيار محفظة + إرسال ملخص الطلب لواتساب الإدارة.
- لوحة تحكم كاملة CRUD (إضافة/تعديل/حذف منتجات، أقسام، محافظ، محتوى، رفع صور للـ bucket).
- المفضلة + الحساب (تسجيل دخول العميل).
- ألبوم "قبل وبعد" القابل للتعديل.

## التفاصيل التقنية (للمرجعية)

```text
src/
├── styles.css                  # ألوان MyCar + خط Cairo + RTL
├── routes/
│   ├── __root.tsx              # html lang="ar" dir="rtl" + QueryClient
│   ├── index.tsx               # الرئيسية
│   ├── shop.tsx
│   ├── product.$id.tsx
│   ├── services.tsx
│   ├── services.$slug.tsx
│   ├── vip.tsx
│   ├── offers.tsx
│   ├── about.tsx
│   ├── contact.tsx
│   ├── admin.tsx
│   └── sitemap[.]xml.ts
├── components/
│   ├── layout/ (TopBar, Header, Footer, BottomNav, MarketingShell)
│   ├── home/ (HeroVideo, CategoryCircles, HotDeals, BestSellers)
│   ├── shop/ (ProductCard, ProductGrid, CategoryFilter)
│   └── ui/ (shadcn)
├── lib/
│   ├── catalog.functions.ts    # getProducts, getCategories, getPackages
│   ├── content.functions.ts    # getSiteContent, getWallets, getServices
│   └── whatsapp.ts             # بناء روابط wa.me
└── assets/                     # الصور المولّدة
```

- TanStack Query: loader → `ensureQueryData` + `useSuspenseQuery`.
- قراءات الكتالوج عامة عبر `createServerFn` + `supabaseAdmin` بـ projection محدّد للأعمدة الآمنة.
- RLS: قراءة عامة للكتالوج، كتابة محصورة (المرحلة 2 تضيف auth حقيقي للأدمن).
- SEO: head() مخصص لكل صفحة، sitemap.xml يشمل كل المسارات.

## بعد موافقتك
سأفعّل Lovable Cloud، أنشئ المهاجرات والبيانات الأولية، أولّد الصور، وأبني كل الصفحات أعلاه. بعدها أبدأ المرحلة 2 (السلة + الدفع + لوحة التحكم الكاملة) برسالتك التالية.
